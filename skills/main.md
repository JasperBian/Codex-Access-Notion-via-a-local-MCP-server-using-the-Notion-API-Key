# Notion Workspace Integration

This plugin connects Codex to your **Notion workspace** via the official Notion MCP server. It provides tools to search, read, create, and update Notion content.

## Getting Started

To use Notion in Codex, you need to **authenticate** with your Notion account. When you first use a Notion command, Codex will prompt you to sign in via OAuth.

## Available Capabilities

The Notion MCP server in this plugin enables:

### Searching
- **Search your workspace**: Find pages and databases by title or content
- **Quick find**: Get a page or database by its title (fuzzy match)

### Reading
- **Retrieve page content**: Get the full content of any page you have access to
- **Query databases**: Filter, sort, and paginate through database entries
- **Read comments and discussions**: Access page comments

### Writing
- **Create pages**: Create new pages under a specified parent
- **Update pages**: Modify page content and properties
- **Append blocks**: Add content blocks (headings, text, lists, code, etc.) to existing pages
- **Insert database rows**: Add new entries to any database

### Properties
- **Read and update properties**: Work with page properties (status, date, select, multi-select, etc.)
- **Property-safe updates**: Update properties without affecting page content

## Quick Commands

When working with Notion, I can help with:

- "Search my Notion for the project plan"
- "Create a new page called 'API Design' under my Engineering docs"
- "Add a task to my Tasks database: 'Fix login bug' — high priority"
- "Query my Projects database for active projects"
- "Save these code exploration notes to my Notion workspace"

## Tips

- Notion databases support different property types — I'll use the right ones for your data
- I can format content with headings, lists, code blocks, callouts, and toggles
- I'll always ask which parent page or database to use when it's not obvious
